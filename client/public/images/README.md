# Logo Gallery

Legg dine logo-bilder her for å gjøre dem tilgjengelige i galleriet på hjemmesiden.

## Filnavn som støttes:
- `logo1.png`
- `logo2.png`
- `logo3.png`
- `logo4.png`
- `logo5.png`
- `logo6.png`

Du kan også bruke `.jpg`, `.jpeg`, `.gif`, `.webp` eller andre bilde-formater.

## Hvordan legge til bilder:
1. Legg bildefilen i denne mappen (`client/public/images/`)
2. Bildet vil automatisk vises i galleriet på hjemmesiden
3. Klikk på bildet for å velge det som logo

## Tips:
- Bruk kvadratiske bilder (samme bredde og høyde) for best resultat
- Anbefalt størrelse: 200x200px eller større
- PNG med transparent bakgrunn fungerer best
